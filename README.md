# Hello World Program

## Introduction
This is a simple Hello World program written in Java.  
The program prints the message:

Hello World

to the console.

---

## Requirements
- Java JDK installed
- Command Prompt / Terminal

---

## File Structure

```text
Hello.java
README.md
```

---

## Program Code

```java
class Hello {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
```

---

## How to Compile

Open terminal or command prompt and run:

```bash
javac Hello.java
```

---

## How to Run

```bash
java Hello
```

---

## Output

```text
Hello World
```
